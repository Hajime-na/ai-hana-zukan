# Release Notes

## v1.0-rc5 — 編集UX改善版

**タグ**: `v1.0-rc5`  
**ベース**: v1.0-rc4 (安定地点)  
**コミット**: `7d51c9b`

### このリリースについて

ポスター編集画面のUXを中心に改善したリリースです。文字スタイルを6種類に再設計し、文字サイズの個別調整スライダーを追加しました。またPC表示での編集作業性を大きく改善し、詳細設定をスクロールしながら常にプレビューを確認できるようになりました。

### 追加・変更内容

#### 文字スタイル6種類

| スタイル | 特徴 |
|---|---|
| 文字なし | 画像のみ。文字を一切表示しない |
| 小さく入れる | 写真の隅に控えめに入れる（初期推奨） |
| 上品文字 | 半透明カード・細フォントで上品な印象 |
| 強めPOP | 太文字・高コントラストでフェア名を訴求 |
| 全面告知 | セール・イベント向けの大きな白文字 |
| 下余白 | 写真の下余白に入れ、画像本体を壊さない |

旧スタイル名（`direct` / `pop-band`）は後方互換マッピングで新名称に自動変換。

#### AI文字配置提案の更新

提案ラベルと設定値を新スタイルに対応：
- **写真を活かす案** ― minimal / タイトル 90%
- **上品文字案** ― elegant / タイトル 100%
- **強めPOP案** ― strong-pop / タイトル 115%

#### 文字サイズ調整スライダー

詳細設定内に3つのスライダーを追加（初期値100%、範囲70〜130%）：
- タイトル文字サイズ
- サブタイトル文字サイズ
- 店舗名・日付文字サイズ

スライダーの値はDOMプレビュー・キャンバス出力・仕上がり確認・注文JSONすべてに反映。リセットボタンでまとめて100%に戻せる。

#### 仕上がり確認画面の整理

- 信頼性のない拡大確認エリア（画質確認・文字品質確認）を非表示化
- 確認チェック項目を右カラムから左カラム（素材警告の直下）へ移動し、視認しやすく整理
- 編集画面の保存ボタンを非表示のまま維持（保存は仕上がり確認画面に集約）

#### 編集画面 sticky プレビュー（PC）

- PC（901px以上）でポスタープレビューが画面上部に固定
- 詳細設定・文字サイズスライダー・テキストオフセットを操作中もプレビューを常時確認可能
- JavaScriptによる scroll 追従実装（`overflow-x: clip` 対応）
- モバイルでは通常レイアウト（固定なし）

### 既存機能への影響

変更なし。以下はすべて正常動作を確認：

- 棚サムネイルクリック・テンプレート選択
- AI販促文・AI修正リクエスト
- PNG保存 / PDF保存 / 仮注文 / サーバー保存
- 注文管理UI（一覧・詳細・ステータス・削除）
- SAMPLE透かし・保存不可制御（poster_allowed: false）

## v1.0-rc3 — カテゴリ棚型ポスターギャラリー対応版

**タグ**: `v1.0-rc3`  
**ベース**: v1.0-rc (公開前整理版)

### このリリースについて

トップページのポスターギャラリーを、検索入力型から**カテゴリ棚型UI**に全面リニューアルしました。ユーザーが検索せずに画面を眺めながら好みのポスターをクリックできるようになりました。

### 追加・変更内容

- **カテゴリ棚型ギャラリー**：春・夏・秋・冬・母の日・ギフト・開店祝いの7カテゴリを常時表示
- **縮小サムネイル一覧**：各カテゴリに最大6枚のポスターサムネイルを横一列で表示
- **即時反映**：サムネイルクリックでポスター編集エリアへ自動スクロール、テキスト・スタイルを自動入力
- **選択中バッジ**：選択中サムネイルをハイライト表示
- **もっと見る**：そのカテゴリ行だけ6枚ずつ追加（ページ全体の切り替えなし）
- **poster_templates.json**：テンプレートデータを外部JSONで管理、後から正式画像に差し替え可能
- **ヒーローエリア**：「花屋さんの販促を、もっと美しく。」キャッチコピー＋CTAボタン
- **花図鑑を補助セクションに移動**：ポスター作成が主役、花図鑑・AI販促文は下部の補助セクション

### 既存機能への影響

変更なし。以下はすべて正常動作：

- PNG保存 / PDF保存 / 仮注文 / サーバー保存 / 注文管理UI
- 正式素材アップロード・権利チェック
- AI販促文・AI修正リクエスト
- 花図鑑検索・詳細表示

### テンプレートデータ仕様

```json
{
  "categories_order": ["春", "夏", ...],
  "templates": [
    {
      "id": "spring-a",
      "short_label": "春A",
      "gradient": "linear-gradient(...)",
      "categories": ["春"],
      "poster_allowed": false,
      "default_main_title": "春の花フェア",
      "default_subtitle": "...",
      "default_note": "...",
      "poster_type": "elegant",
      "poster_design": "photo-full",
      "poster_position": "bottom-left"
    }
  ]
}
```

> `poster_allowed: false` のテンプレートは仮表示用です。正式素材をアップロードすることでPNG/PDF保存が有効になります。
