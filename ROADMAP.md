# Roadmap

v1.0 でポスター制作・注文保存・Prio発注管理・Slack共有管理が完成。  
v1.1 では実運用に向けた「お客様情報」「配送先」「支払い管理」「注文確認」を整備する。  
v1.2 では花屋専用サービスから「想い出」「カフェ・店舗」の2カテゴリを扱うポスター作成サービスへ広げる。花・ギフトは独立カテゴリではなく、カフェ・店舗ポスターの用途の一部として扱う。

---

## v1.2 — 2カテゴリ展開（想い出 / カフェ・店舗）

### 目的

Hana Poster AI を花屋専用ではなく、以下の2カテゴリを扱うポスター作成サービスへ広げる。既存の注文管理・見積もり・支払い・入稿PDF生成・Prio発注管理は維持したまま、入口とヒアリング〜制作フローをカテゴリごとに最適化する。

### カテゴリと基本フロー

| カテゴリ | 基本フロー |
|---|---|
| 想い出ポスター | 写真アップロード → 高画質化/補正 → 文字入れ → 確認 → 発注 |
| カフェ・店舗ポスター | 用途・雰囲気ヒアリング → 提案作成 → 選択/修正 → 確認 → 発注（花・ギフトの告知も本カテゴリで扱う） |

既存の花素材ギャラリー（作例エリア・ポスターテンプレート）は削除せず、カフェ・店舗ポスター向けの「季節・花素材」として下位に位置づける。

### v1.2-rc1 — トップ画面に3つの入口カードを追加（公開済み・履歴）

トップ画面ヒーロー直下に「想い出ポスター」「カフェ・店舗ポスター」「花・ギフトポスター」の3枚の入口カードを追加。GitHubタグ・リリース済みのため、この内容は変更しない。

### v1.2-rc2 — 2カテゴリへ再編（完了）

花・ギフトを独立カテゴリから外し、カフェ・店舗ポスターに統合。入口カードを「想い出ポスター」「カフェ・店舗ポスター」の2枚に変更。ヒーロー文言（Memory & Shop Poster、「想い出を、飾れる一枚に。」）・作例エリアの説明文もこの方針に合わせて更新。

### 今後（未着手）

- [ ] 想い出ポスター専用フロー：写真アップロード後の高画質化・補正処理
- [ ] カフェ・店舗ポスター専用フロー：ヒアリング内容から提案ポスターを自動生成
- [ ] カテゴリごとのギャラリー・テンプレート整理

## v1.1-rc1 — お客様情報・配送先・注文確認

### 目的

注文ごとにお客様情報・配送先・希望納期を記録し、注文確定前チェックを行えるようにする。

### お客様情報フィールド

| フィールド | 備考 |
|---|---|
| お名前 | |
| 店舗名 | |
| 担当者名 | |
| メールアドレス | |
| 電話番号 | |
| 郵便番号 | |
| 住所 | 請求先住所 |
| 配送先住所 | 未記入の場合は住所と同一 |
| 希望納期 | |
| 備考 | |
| 領収書宛名 | 空欄可 |

### 注文確定前チェック項目

注文確定ボタンの前に表示するチェックリスト（全チェックで確定ボタン活性化）：

- [ ] ポスターデザインを確認しました
- [ ] 印刷サイズ・用紙・配送種別を確認しました
- [ ] 配送先住所を確認しました
- [ ] 希望納期を確認しました
- [ ] 領収書宛名を確認しました（不要な場合はチェック）

## v1.1-rc2 — 支払い方法・入金ステータス

### 目的

注文ごとに支払い方法と入金状態を管理し、管理画面でステータス変更・Slack通知できるようにする。

### 支払い方法

- 銀行振込
- PayPay
- Square決済リンク
- その他

### 支払い状態

| ステータス | 意味 |
|---|---|
| 未請求 | 注文確定前・請求書未発行 |
| 請求済み | 請求書送付済み・入金待ち |
| 入金待ち | 入金確認前 |
| 入金済み | 入金確認済み |
| 返金 | 返金処理済み |
| キャンセル | 注文キャンセル |

### 管理画面

- 支払い状態を管理画面からドロップダウンで変更
- ステータス変更時に Slack 通知（発注管理通知と同チャンネル）

## v1.1-rc3 — 見積もり・請求メモ

### 目的

注文ごとに費用内訳を記録し、お客様向け案内文をコピーできるようにする。

### フィールド

| フィールド | 備考 |
|---|---|
| 見積金額 | 合計と一致しない場合に使用 |
| 印刷費 | Prioへの支払額 |
| 制作費 | デザイン料 |
| 送料 | |
| 合計 | 自動計算（税込） |
| 管理メモ | 内部用メモ（お客様には非表示） |
| お客様向け案内文 | コピー用テキスト生成 |

### お客様向け案内文コピー

管理画面にコピーボタンを設置。以下の形式でクリップボードにコピー：

```
【ご注文確認】
ご注文ありがとうございます。

■ 印刷サイズ：{size}
■ 用紙：{paper}
■ 配送種別：{delivery}
■ 合計：¥{total}（税込）
■ お支払い方法：{payment_method}
■ 希望納期：{due_date}

印刷確認ID：{print_check_id}
```

## v1.1-rc4 — 実印刷テスト

### 目的

Prio への実発注前に、自宅宛てでテスト印刷を行い品質を確認する。

### 確認項目

- [ ] 自宅宛てにテスト発注（A2 / マット紙 / 余裕便）
- [ ] 受け取り後、色味を画面と比較
- [ ] 紙質・厚み・光沢の確認
- [ ] 梱包状態の確認（折れ・傷）
- [ ] 納期の確認（注文からの日数）
- [ ] PDF右下の印刷確認IDが印字されているか確認

### 完了条件

上記すべて確認済みで、品質に問題がないこと。問題があれば Prio サポートに連絡し、rc4 を再実施。
