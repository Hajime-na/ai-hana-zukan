# AI花図鑑｜花屋さんの販促・ポスター作成ツール

花屋さん向けに、花の情報確認・販促文作成・ポスター文字入れ・正式素材アップロード・PNG保存・仮注文確認までできる試作Webアプリです。

---

## 主な機能

### 花図鑑
- 花名・読み方・英名で検索
- 花詳細表示（育て方・花言葉など）
- 複数写真の切り替え
- 出典・ライセンス表示

### AI販促文作成
- 店頭POP文の生成
- SNS（Instagram）投稿文の生成

### ポスター文字入れ
- 正式素材（ユーザーアップロード画像）への文字入れ
- 素材の権利チェック（外部写真はPNG保存・発注不可）
- 仕上がり確認プレビュー
- PNG保存
- 仮注文確認

### 仮注文・ファイル紐付け（v0.4〜）
- 仮注文時に `order_id`（YYYYMMDD_HHMMSS形式）を自動生成
- 注文内容をJSONファイルとしてダウンロード可能
- PNGを同じ `order_id` 付きファイル名で保存可能
- JSON内に `png_filename` を記録し、後からJSONとPNGを対応付け可能

---

## 画像素材の扱い

| 種別 | 用途 | PNG保存 | 仮注文 |
|------|------|---------|--------|
| 外部写真（Unsplash等） | 花図鑑の参考表示のみ | 不可 | 不可 |
| ユーザーアップロード素材 | ポスター文字入れ | 可 | 可 |

> 本番公開前に、写真内容・ライセンス表記が正しいか必ず確認してください。

---

## 画像の使い方（2モード）

- **写真に文字を入れる** ― アプリ側で文字帯・タイトル・店舗名・日付を重ねます
- **画像をそのまま使う** ― 写真をそのままポスター素材として使用します

---

## 対応サイズ

| 値 | 説明 |
|----|------|
| A4縦（デフォルト） | 210 × 297 mm |
| A4横 | 297 × 210 mm |
| 元画像比率 | アップロード画像のアスペクト比をそのまま使用 |
| 正方形 | 1:1 |
| 16:9 | ワイド横長 |

---

## ファイル名仕様（order_id 紐付け）

仮注文時に生成される `order_id` を起点に、JSONとPNGが対応付けられます。

```
order_id : 20260601_194358
JSON     : order_20260601_194358.json
PNG      : poster_20260601_194358.png
```

---

## 仮注文JSONに含まれる項目

| フィールド | 内容 |
|------------|------|
| `order_id` | 注文ID（YYYYMMDD_HHMMSS） |
| `created_at` | 注文日時（ISO 8601） |
| `flower_name` | 選択した花の名前 |
| `image_usage` | 画像の使い方（文字入れ／そのまま） |
| `poster_size` | ポスターサイズ |
| `poster_rotation` | ポスターの向き回転 |
| `image_fit` | 画像フィットモード |
| `image_position` | 画像位置 |
| `image_zoom` | 画像ズーム |
| `image_offset_x` | 画像横オフセット |
| `image_offset_y` | 画像縦オフセット |
| `main_title` | メインタイトル |
| `subtitle` | サブタイトル |
| `shop_name` | 店舗名 |
| `period` | 期間 |
| `note` | 補足文 |
| `text_style` | テキストスタイル |
| `text_position` | テキスト位置 |
| `material_status` | 素材の権利状態メモ |
| `poster_allowed` | ポスター利用可否 |
| `png_filename` | 対応するPNGファイル名 |
| `checks` | 確認チェック項目5項目の状態 |
