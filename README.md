# AI花図鑑｜花屋さんの販促・ポスター作成ツール

花屋さん向けに、花の情報確認・販促文作成・ポスター文字入れ・正式素材アップロード・PNG保存・仮注文確認・仮注文履歴一覧までできる試作Webアプリです。

---

## 主な機能

### 花図鑑
- 花名・読み方・英名で検索
- 花詳細表示（育て方・花言葉など）
- 複数写真の切り替え
- 出典・ライセンス表示

### AI販促文作成
- 店頭POP文の生成
- SNS（Instagram）投稿文の生成

### ポスター文字入れ
- 正式素材（ユーザーアップロード画像）への文字入れ
- 素材の権利チェック（外部写真はPNG保存・発注不可）
- 仕上がり確認プレビュー
- PNG保存
- 仮注文確認

### 仮注文・ファイル紐付け（v0.4〜）
- 仮注文時に `order_id`（YYYYMMDD_HHMMSS形式）を自動生成
- 注文内容をJSONファイルとしてダウンロード可能
- PNGを同じ `order_id` 付きファイル名で保存可能
- JSON内に `png_filename` を記録し、後からJSONとPNGを対応付け可能

### 仮注文履歴一覧（v0.5〜）
- 仮注文するたびに履歴をブラウザの localStorage に自動保存
- 履歴を新しい順に一覧表示
- 各履歴からJSONを再保存可能
- 個別履歴の削除
- 全履歴の一括削除
- ページ再読み込み後も履歴を保持

## 画像素材の扱い

| 種別 | 用途 | PNG保存 | 仮注文 |
|------|------|---------|--------|
| 外部写真（Unsplash等） | 花図鑑の参考表示のみ | 不可 | 不可 |
| ユーザーアップロード素材 | ポスター文字入れ | 可 | 可 |

> 本番公開前に、写真内容・ライセンス表記が正しいか必ず確認してください。

---

## 画像の使い方（2モード）

- **写真に文字を入れる** ― アプリ側で文字帯・タイトル・店舗名・日付を重ねます
- **画像をそのまま使う** ― 写真をそのままポスター素材として使用します

---

## 対応サイズ

| 値 | 説明 |
|----|------|
| A4縦（デフォルト） | 210 × 297 mm |
| A4横 | 297 × 210 mm |
| 元画像比率 | アップロード画像のアスペクト比をそのまま使用 |
| 正方形 | 1:1 |
| 16:9 | ワイド横長 |

---

## ファイル名仕様（order_id 紐付け）

仮注文時に生成される `order_id` を起点に、JSONとPNGが対応付けられます。

```
order_id : 20260601_194358
JSON     : order_20260601_194358.json
PNG      : poster_20260601_194358.png
```

---

## 仮注文履歴の注意事項

- 履歴はこのブラウザ内の localStorage に保存されます
- サーバーには保存されません
- ブラウザのデータ削除や別端末では履歴は共有されません
- 本番運用ではサーバー保存・注文管理画面が必要です

- JSONを再保存 → ファイル名・`order_id` が一致することを確認
- 個別削除 → 対象のみ削除されることを確認
- 全削除 → 空メッセージが表示されることを確認
- 全削除後に再読み込みしても空のままであることを確認
