# AI花図鑑｜花屋さんの販促・ポスター作成ツール

花屋さん向けに、花の情報確認・販促文作成・ポスター文字入れ・正式素材アップロード・PNG保存・仮注文確認までできる試作Webアプリです。

---

## 主な機能

### 花図鑑
- 花名・読み方・英名で検索
- 花詳細表示（育て方・花言葉など）
- 複数写真の切り替え
- 出典・ライセンス表示

### AI販促文作成
- 店頭POP文の生成
- SNS（Instagram）投稿文の生成

### ポスター文字入れ
- 正式素材（ユーザーアップロード画像）への文字入れ
- 素材の権利チェック（外部写真はPNG保存・発注不可）
- 仕上がり確認プレビュー
- PNG保存
- 仮注文確認

---

## 画像素材の扱い

| 種別 | 用途 | PNG保存 | 仮注文 |
|------|------|---------|--------|
| 外部写真（Unsplash等） | 花図鑑の参考表示のみ | 不可 | 不可 |
| ユーザーアップロード素材 | ポスター文字入れ | 可 | 可 |

> 本番公開前に、写真内容・ライセンス表記が正しいか必ず確認してください。

---

## 画像の使い方（2モード）

- **写真に文字を入れる** ― アプリ側で文字帯・タイトル・店舗名・日付を重ねます
- **画像をそのまま使う** ― 写真をそのままポスター素材として使用します

---

## 対応サイズ

| 値 | 説明 |
|----|------|
| A4縦（デフォルト） | 210 × 297 mm |
| A4横 | 297 × 210 mm |
| 元画像比率 | アップロード画像のアスペクト比をそのまま使用 |
| 正方形 | 1:1 |
| 16:9 | ワイド横長 |

## 起動方法

```bash
pip install fastapi uvicorn
uvicorn main:app --reload
```

ブラウザで開く：

```
http://127.0.0.1:8000
```
